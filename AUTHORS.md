## Contributors

We welcome contributions from everyone. All contributors will be recognized here.

To become a contributor:

1. Fork the repository
2. Make your changes
3. Submit a pull request
4. Once merged, your name will be added to this list

### How to Add Yourself

If your contribution has been merged, feel free to add yourself to this list via a pull request with the following format:

```markdown
- **Your Name** — Brief description of contribution
  - GitHub: [@username](https://github.com/username)
```
